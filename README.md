# lioss

License Identification tool for OSS project.

## Usage

```
$ lioss [OPTIONS] <PROJECTS...>
OPTIONS
       --dbpath <DBPATH>    specifying database path.
    -h, --help              print this message.
PROJECTS
    project directories, and/or archive files contains LICENSE file.
```
